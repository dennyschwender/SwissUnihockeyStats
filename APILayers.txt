Indexing levels
SEASONS: GET https://api-v2.swissunihockey.ch/api/seasons -> Will give all the seasons
    CLUBS: For each season, GET https://api-v2.swissunihockey.ch/api/clubs?season=:seasonYear
        CLUB TEAMS: For each club GET https://api-v2.swissunihockey.ch/api/teams?mode=by_club&season=:seasonYear&club_id=:clubID
            TEAM DETAILS: For each team: GET https://api-v2.swissunihockey.ch/api/teams/:team_id
            TEAM PLAYERS: For each team: GET https://api-v2.swissunihockey.ch/api/teams/:team_id/players
            TEAM STATS: For each team: GET https://api-v2.swissunihockey.ch/api/teams/:team_id/statistics
                PLAYER: GET https://api-v2.swissunihockey.ch/api/players/:player_id
                PLAYER STATS: GET https://api-v2.swissunihockey.ch/api/players/:player_id/statistics
                PLAYER GAMES OVERVIEW: GET https://api-v2.swissunihockey.ch/api/players/:player_id/overview
    LEAGUES: For each sason: GET https://api-v2.swissunihockey.ch/api/leagues?season=:seasonYear
        LEAGUE GROUP: For each League: GET https://api-v2.swissunihockey.ch/api/groups?season=:seasonYear&league=:leagueID&game_class=gameClassID
            LEAGUE VISITORS: For each group and game class: https://api-v2.swissunihockey.ch/api/teams/visitors?season=:seasonYear&league=:leagueID&game_class=gameClassID
            LEAGUE GROUP TEAMS: For each group and game class: GET https://api-v2.swissunihockey.ch/api/teams?season=:seasonYear&league=:leagueID&game_class=gameClassID
            GROUP RANKING REGULAR SEASON: For each group and game class: GET https://api-v2.swissunihockey.ch/api/rankings?season=:seasonYear&league=:leagueID&game_class=gameClassID
            GAMES OF A GROUP: For each group and game class: GET https://api-v2.swissunihockey.ch/api/games?mode=list&season=:seasonYear&league=:leagueID&game_class=gameClassID
                GAME DETAILS: GET https://api-v2.swissunihockey.ch/api/games/:game_id
                GAME SUMMARY: GET https://api-v2.swissunihockey.ch/api/games/:game_id/summary
                GAME LINEUP: GET https://api-v2.swissunihockey.ch/api/games/:game_id/teams/:is_home/players
                GAME EVENTS: GET https://api-v2.swissunihockey.ch/api/game_events/:game_id
