# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project setup
- SwissUnihockeyClient API wrapper
- Support for 13 public API endpoints
- Retry logic with exponential backoff
- Configuration file support (config.ini)
- Data caching structure (data/raw, data/processed)
- Example scripts for data fetching
- Test script for API verification
- Comprehensive documentation (README, GETTING_STARTED, FEATURE_IDEAS)
- API usage examples and code snippets
- Support for 4 locales (en, de-CH, fr-CH, it-CH)

### Features
- Fetch all 346 Swiss Unihockey clubs
- Access 50+ league/game class combinations
- Historical data from 31 seasons (1995/96 - 2026/27)
- League rankings/standings
- Top scorers statistics
- Match schedules and results
- Player and team information
- Game events (play-by-play)
